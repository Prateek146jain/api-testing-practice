# API Testing Practice

Hello Every one 

This project contains Postman collection and Newman reposts for practicing API Testing 

# what is inside this repo??

1.Postman Collection Json file 
2.Newman Html report 
3.Test data file (Not needed)

# Tools Used
1. Postman 
2. Newman 


# how to Run this 

1. First intall node.js
2. install newman 
    npm install -g newman 
3. Run the collection 
    newman run (YourfileName).json 

# Note-> if you run this command "newman run (YourfileName).json -r htmlextra", you can see the html report.
